# StudyTree

参考書の構造をそのまま保存して、構造ごと再現する受験勉強アプリ。

## 機能

- **学習項目登録** — タイトル・科目・学習日・小タイトル（数制限なし）
- **ツリー表示** — 科目ごとに折りたたみ表示
- **想起モード** — 内容を隠してタップで1つずつ表示
- **白紙再現モード** — タイトルだけ見て小タイトルを思い出す
- **自己評価** — 完璧 / あいまい / 思い出せなかった
- **統計** — 登録数・想起回数・苦手項目の確認

データは `localStorage` に保存。オフライン対応。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## デプロイ（Vercel）

1. このリポジトリを GitHub に push
2. [vercel.com](https://vercel.com) でリポジトリをインポート
3. 設定はデフォルトのままでデプロイ
